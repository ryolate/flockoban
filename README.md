# Flockoban

Automatic solver for https://le-slo.itch.io/flockoban

## Set up

`apt install pypy3`

`pypy3 -m pip install sortedcontainers`

## Usage

`cat 0820.in | python solver.py`

## Development

Format code with autopep8 (VS Code default).
